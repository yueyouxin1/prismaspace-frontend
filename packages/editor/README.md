# editor

Dedicated editor package for reusable code/markdown editing capabilities.

- `src/components/monaco-editor/*`: Monaco based editor family.
- `src/components/monaco-editor/md-editor/*`: Markdown expression editor built on `MonacoEditor`.
- `src/components/codemirror-editor/*`: Base CodeMirror 6 code editor.
- `src/components/codemirror-editor/codemirror-md-editor/*`: Markdown expression editor built on `CodeMirrorEditor`.
- `src/components/param-schema-editor/*`: Param Schema editor suite (regular/professional), core tree ops, import/export and runtime validation.
- `src/plugin.ts`: global editor component plugin (`EditorUiPlugin`).
