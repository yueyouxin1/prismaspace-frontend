# @prismaspace/editor

通用编辑器能力包。

## 公开能力

- Monaco 编辑器族
- CodeMirror 编辑器族
- Markdown 表达式编辑器
- Param Schema 编辑器
- `EditorUiPlugin`

## 主要导出

- `MonacoEditor`
- `MdEditor`
- `CodeMirrorEditor`
- `CodeMirrorMdEditor`
- `ParamSchemaRegularEditor`
- `ParamSchemaProfessionalEditor`
- `EditorUiPlugin`

## 适用场景

- 资源工作台中的 DSL/Prompt/Schema 编辑
- SaaS shell 中的编辑能力复用
- 第三方项目中的编辑器嵌入
