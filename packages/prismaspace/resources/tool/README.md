# @prismaspace/tool

Tool 资源公开包。

## 公开入口

- `@prismaspace/tool`
  - 主入口：`ToolWorkbench`
- `@prismaspace/tool/main`
  - `ToolWorkbench`
- `@prismaspace/tool/workbench`
  - `ToolWorkbench`
- `@prismaspace/tool/runtime`
  - schema adapter 与 draft composable
- `@prismaspace/tool/types`
  - Tool 公共类型

## 当前能力

- Tool 参数 schema 编辑
- 保存 / 运行 / 发布
- 运行结果反馈
