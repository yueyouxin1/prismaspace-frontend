# @prismaspace/uiapp

UiApp 资源公开包。

## 公开入口

- `@prismaspace/uiapp`
  - 主入口：`UiAppWebRenderer`
- `@prismaspace/uiapp/main`
  - `UiAppWebRenderer`
- `@prismaspace/uiapp/workbench`
  - `UiAppWorkbench`
- `@prismaspace/uiapp/runtime`
  - 预留运行时入口
- `@prismaspace/uiapp/types`
  - 预留类型入口

## 当前能力

- `UiAppWebRenderer`
  - 包主入口，负责加载并渲染指定页面 DSL
- 页面列表浏览
- 页面 DSL 查看
- 发布能力
