# @prismaspace/knowledge

Knowledge 资源公开包。

## 公开入口

- `@prismaspace/knowledge`
  - 主入口：`KnowledgeWorkbench`
- `@prismaspace/knowledge/main`
  - `KnowledgeWorkbench`
- `@prismaspace/knowledge/workbench`
  - `KnowledgeWorkbench` / `KnowledgeBaseIdeWorkbench`
- `@prismaspace/knowledge/runtime`
  - store factory 与 normalizers
- `@prismaspace/knowledge/types`
  - knowledge 公共类型

## 当前能力

- `KnowledgeWorkbench`
- `KnowledgeBaseIdeWorkbench`
- `createKnowledgeWorkbenchStore`
- `knowledge-normalizers`

## 典型场景

- 知识库文档管理
- chunk 检查与编辑
- 检索测试
- 配置与发布
