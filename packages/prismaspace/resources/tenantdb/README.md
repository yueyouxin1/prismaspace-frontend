# @prismaspace/tenantdb

TenantDB 资源公开包。

## 公开入口

- `@prismaspace/tenantdb`
  - 主入口：`TenantDbWorkbench`
- `@prismaspace/tenantdb/main`
  - `TenantDbWorkbench`
- `@prismaspace/tenantdb/workbench`
  - `TenantDbWorkbench`
- `@prismaspace/tenantdb/runtime`
  - TenantDB 运行时类型
- `@prismaspace/tenantdb/types`
  - TenantDB 公共类型

## 当前能力

- 表结构编辑
- 查询构造器
- 行级增删改查
- SQL 查询面板
- 发布能力
